# DBMS Coursework – Web Application Project

This repository contains my university Database Management Systems (DBMS) coursework project implemented using PHP and MySQL.

## Project Overview
The project is a basic web-based application developed to demonstrate core DBMS concepts such as database connectivity, data handling, and backend integration using PHP.

The application was developed and tested in a local server environment using XAMPP and phpMyAdmin.

## Technologies Used
- PHP
- MySQL
- HTML
- XAMPP (Apache Server)
- phpMyAdmin

## Files Included
- index.php – Main application entry point
- applications.html – Web interface page
- bitnami.css – Styling file
- favicon.ico – Application icon

## DBMS Concepts Applied
- Database connection using PHP
- Data storage and retrieval
- Backend–frontend integration
- Relational database usage

## Purpose
This project was completed as part of the Database Management Systems course for academic learning and practice purposes.
